CSSの概要

CSSの基本ルール

基本書式

セレクタ・プロパティ・値の基本書式に沿って「どこの」「何を」「どうする」かを定義する

セレクタ { プロパティ: 値 }
【例】
h1 { color: #ff0000; }

CSSでの色指定

上記サンプルに載せた方法以外でも #ff0000などで表現する16進数、10進数で表現するRGB値、不透明度も指定できるRGBaの設定、redなどのカラーネームでの指定もできる。

CSSで使う単位

相対単位と絶対単位で指定する方法がある。

相対単位

px
モニタの画素（ピクセル）を1とする単位
%
パーセントで割合を指定
em
親要素の小文字xの高さを1とする単位
rem
ルート要素の大文字Mの高さを1とする単位
vw
viewportの幅の1/100を1とする単位
vh
viewportの高さの1/100を1とする単位

所感: emとremを見かけることが多い気がするけど、remで定義した方が基準となるサイズが一意なので迷子にはなりにくそう

絶対単位

pt
ポイント。1pt = 1/72in
pc
パイカ。1pc = 12pt
mm
ミリメートル
cm
センチメートル
in
インチ。 1in = 2.54cm

所感: viewportを考慮した記述には向いていなさそうなので、帳票など物理的に出力するものに使用するのが良さそう
